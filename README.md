# 🎓 AI Study Planner & Career Coach

An AI-powered full-stack web application that helps students organize their learning journey, analyze their skills, prepare for interviews, and receive personalized career guidance.

Built with the **MERN Stack (MongoDB, Express.js, React, Node.js)** and enhanced with AI services for intelligent recommendations and planning.

# 👨‍💻 Getting Started

## Clone the Repository

```bash
git clone https://github.com/your-username/AI-Study-Planner.git
```

## Frontend

```bash
cd ai-study-planner-client
npm install
npm run dev
```

## Backend

```bash
cd ai-study-planner-server
npm install
npm run dev
```
